# README

This project contains data to practice quality control (QC) and learn genome-wide 
association study analyses. 

Contents

File/Directory			Description
readme.txt				This readme.
gwas					Example GWAS study, already cleaned.
rawdata					Example 'raw' data to learn QC and GWAS.
ref_1kg_phase1_all		Prepared 1000G reference data.
scripts					Some useful scripts.
support					Contains a file with a list of problematic regions.
